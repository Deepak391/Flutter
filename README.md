 ## How to Use 

```
- Clone this repository:
$ git clone -b SimpleCalculator https://github.com/Deepak391/Flutter

- Enter in directory:
$ cd Flutter

- For install dependencies:
$ flutter packages get

- Run the app: 
$ flutter run 
```
